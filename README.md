# TFI Lembas

Documentación técnica del TFI **Lembas**, una aplicación web interna para una dietética de una sola sucursal.

## Descripción

Este repositorio contiene la documentación modular del sistema, organizada por áreas para facilitar lectura, mantenimiento y trazabilidad.

## Estructura

- `Documentation/` → documentación principal del proyecto
  - `00 - Vision general.md`
  - `01 - Contexto y problema.md`
  - `02 - Alcance del sistema.md`
  - `03 - Requisitos/`
  - `04 - Dominio/`
  - `05 - Casos de uso/`
  - `06 - Arquitectura/`
  - `07 - API/`
  - `08 - UX UI/`
  - `09 - Testing y calidad/`
  - `10 - Planificacion/`
  - `11 - Anexos/`
  - `attachments/` (imágenes de apoyo)

## Navegación recomendada

1. `Documentation/00 - Vision general.md`
2. `Documentation/01 - Contexto y problema.md`
3. `Documentation/02 - Alcance del sistema.md`

Luego continuar por módulos según necesidad:

- Requisitos
- Dominio
- Casos de uso
- Arquitectura
- API
- UX/UI
- Calidad
- Planificación
- Anexos

## Convenciones

- La documentación está dividida en archivos pequeños y especializados.
- La trazabilidad de requisitos se mantiene en:
  - `Documentation/03 - Requisitos/03.3 - Matriz de trazabilidad.md`
- Se usan rutas relativas para imágenes en `Documentation/attachments/`.

## Requisitos para abrir la documentación

- Cualquier editor Markdown.
- Recomendado: Obsidian para aprovechar enlaces wiki (`[[...]]`).
