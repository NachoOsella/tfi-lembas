# Actores

| Actor | Descripcion | Necesidades principales |
|---|---|---|
| Administradora/Dueña | Responsable del negocio y usuaria con maxima autoridad funcional. | Gestionar productos, proveedores, precios, stock, caja, reportes, auditoria y usuarios. |
| Empleado | Usuario operativo del local. | Registrar ventas, consultar productos, registrar tareas permitidas y cerrar su turno. |
| Sistema | Componente automatico que valida, calcula, audita y genera alertas. | Mantener consistencia entre venta, stock, caja y auditoria. |

## Permisos resumidos

| Funcionalidad | Administradora | Empleado |
|---|---:|---:|
| Ver home administrativo | Si | No |
| Ver home operativo | Si | Si |
| Gestionar usuarios | Si | No |
| Crear/editar productos | Si | No, salvo permiso |
| Consultar productos y precios | Si | Si |
| Ver costos y margenes | Si | No |
| Registrar venta | Si | Si |
| Modificar cantidad en venta | Si | Si |
| Modificar precio en venta | Si | Solo con permiso |
| Anular venta | Si | Solo con permiso |
| Registrar consumo interno | Si | Si, si esta habilitado |
| Registrar merma | Si | Solo con permiso |
| Registrar compras simples | Si | Solo con permiso |
| Importar listas de precios | Si | No |
| Aprobar cambios de precio | Si | No |
| Cerrar turno propio | Si | Si |
| Cerrar caja diaria | Si | No |
| Ver auditoria | Si | No |
