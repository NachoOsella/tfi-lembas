# Glosario del dominio

| Termino | Definicion |
|---|---|
| Administradora/Dueña | Usuario con acceso completo al sistema y decisiones comerciales. |
| Empleado | Usuario operativo con permisos limitados para venta, consulta, tareas y cierre de turno. |
| Producto interno | Producto normalizado dentro de Lembas. |
| Producto de proveedor | Nombre, codigo o presentacion usado por un proveedor para referirse a un producto. |
| ProductoProveedor | Mapeo entre producto interno y producto de proveedor. |
| Codigo de barras | Identificador escaneable opcional del producto. |
| Lote | Grupo de unidades de un producto, normalmente asociado a ingreso y vencimiento. |
| Vencimiento | Fecha limite de consumo o venta recomendada/permitida. |
| FEFO | First Expired, First Out: primero vence, primero sale. |
| Stock minimo | Umbral a partir del cual el sistema considera que falta reponer. |
| Stock ideal | Cantidad objetivo usada para sugerir reposicion. |
| Movimiento de stock | Registro de entrada o salida de stock con tipo, cantidad, usuario y referencia. |
| Venta mostrador simplificada | Registro interno de venta no fiscal, con impacto en stock y caja. |
| Precio base historico | Precio de venta vigente al momento de agregar el producto a una venta. |
| Precio aplicado | Precio efectivamente cobrado en una linea de venta. |
| Oferta temporal | Promocion por producto y periodo, por porcentaje o precio fijo. |
| Consumo interno | Producto consumido dentro del negocio sin venta ni ingreso de caja. |
| Merma | Perdida por rotura, deterioro u otra causa no comercial. |
| Caja | Registro interno de ingresos, egresos y cierres operativos. |
| Turno de caja | Periodo operativo asociado a un usuario o jornada. |
| Compra simple | Registro minimo de productos esperados y recibidos de un proveedor; al confirmarse ingresa stock por la cantidad recibida real. |
| Detalle de compra | Linea de una compra simple que compara cantidad/costo esperado contra cantidad/costo recibido. |
| Detalle de compra lote | Division manual de una linea recibida en uno o mas lotes con cantidad, vencimiento opcional y lote opcional. |
| Cantidad esperada | Cantidad precargada antes de recibir mercaderia. |
| Cantidad recibida | Cantidad real controlada al llegar el proveedor; es la que impacta stock. |
| Lista de precios | Archivo, texto o carga manual enviada por proveedor con costos. |
| Costo proveedor | Valor de compra informado por un proveedor. |
| Precio de venta | Valor cobrado al cliente. |
| Margen | Diferencia relativa entre costo y precio de venta. |
| Precio sugerido | Precio calculado por reglas de margen y redondeo. |
| Etiqueta pendiente | Marca que indica que debe imprimirse o actualizarse la etiqueta visible. |
| Auditoria | Registro trazable de acciones criticas. |
| POS fiscal | Sistema de punto de venta con obligaciones fiscales; esta fuera del alcance de Lembas. |
| OCR | Reconocimiento optico de caracteres; fuera del MVP. |
