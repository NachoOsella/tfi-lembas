# Base de datos

## Motor

PostgreSQL por integridad referencial, transacciones, indices, restricciones y capacidad para consultas operativas.

## Criterios de diseño

- Claves primarias tecnicas.
- Claves foraneas para relaciones criticas.
- Baja logica en entidades con historial.
- Estados controlados en entidades transaccionales.
- Columnas de auditoria: `created_at`, `updated_at`, `created_by`, `updated_by` cuando corresponda.
- Migraciones versionadas con Flyway o Liquibase.

## Indices recomendados

| Tabla | Indice | Motivo |
|---|---|---|
| products | barcode | Busqueda por scanner. |
| products | name | Busqueda manual. |
| product_suppliers | product_id, supplier_id | Mapeo y comparacion. |
| stock_lots | product_id, expiration_date | FEFO y vencimientos. |
| stock_movements | product_id, created_at | Historial. |
| sales | created_at, user_id, status | Ventas por turno/dia. |
| cash_movements | shift_id, created_at, type | Caja y cierres. |
| purchases | supplier_id, purchase_date, status | Consulta de compras por proveedor y fecha. |
| purchase_items | purchase_id, product_id | Detalle de productos esperados/recibidos. |
| purchase_item_lots | purchase_item_id, expiration_date | Division de recepcion por lotes y vencimientos. |
| supplier_price_lists | supplier_id, list_date | Historial de listas. |
| supplier_price_list_items | list_id, recognition_status | Revision de items. |
| audit_logs | entity, entity_id, created_at | Trazabilidad. |

## Restricciones clave

| Restriccion | Aplicacion |
|---|---|
| Codigo de barras unico entre activos | Evita ambiguedad en scanner. |
| Cantidades mayores que cero en egresos normales | Evita movimientos invalidos. |
| Precio de venta no negativo | Integridad comercial. |
| Estados validos por enum o check | Control de ciclos de vida. |
| Foreign keys en movimientos | Trazabilidad a entidades origen. |

## Migraciones

Las migraciones deben crear primero tablas base (usuarios, productos, proveedores), luego tablas transaccionales (stock, ventas, caja), luego historiales, alertas y auditoria.
