# Roles y pantallas

## Administradora/Dueña

Debe poder ver indicadores y acciones de gestion:

- ventas del dia;
- caja actual;
- stock bajo;
- productos proximos a vencer;
- listas de precios pendientes;
- compras simples recientes;
- etiquetas pendientes;
- reposicion sugerida;
- reportes;
- auditoria;
- accesos a productos, proveedores, stock, caja y precios.

## Empleado

Debe tener una vista operativa simple:

- boton principal `Nueva venta`;
- consulta de producto/precio/stock;
- ventas de su turno;
- productos a revisar;
- proximos vencimientos visibles sin costos;
- consumo interno si esta habilitado;
- cierre de turno.

No debe ver por defecto costos, margenes, reportes globales, configuracion, auditoria completa ni aplicacion masiva de precios.
