# Flujos de usuario

## Empleado: venta rapida

```mermaid
flowchart LR
    A[Home empleado] --> B[Nueva venta]
    B --> C[Escanear producto]
    C --> D[Agregar item]
    D --> E{Mas productos?}
    E -->|Si| C
    E -->|No| F[Confirmar]
    F --> G[Venta confirmada]
```

## Administradora: actualizar precios desde lista

```mermaid
flowchart TD
    A[Home admin] --> B[Proveedores]
    B --> C[Cargar lista]
    C --> D[Revisar items]
    D --> E[Comparar costos]
    E --> F[Ver precio sugerido]
    F --> G[Aprobar seleccion]
    G --> H[Etiquetas pendientes]
```

## Empleado: cierre de turno

```mermaid
flowchart TD
    A[Home empleado] --> B[Cierre de turno]
    B --> C[Resumen de ventas y caja]
    C --> D[Ingresar saldo contado]
    D --> E[Confirmar diferencia]
    E --> F[Turno cerrado]
```

## Administradora: control de vencimientos

```mermaid
flowchart TD
    A[Home admin] --> B[Productos proximos a vencer]
    B --> C[Ver lotes]
    C --> D{Accion}
    D --> E[Crear oferta]
    D --> F[Registrar merma/vencimiento]
    D --> G[Dejar en observacion]
```


## Administradora: registrar y recibir compra simple

```mermaid
flowchart TD
    A[Home admin] --> B[Compras simples]
    B --> C[Nueva compra]
    C --> D[Seleccionar proveedor]
    D --> E[Agregar productos esperados]
    E --> F[Indicar cantidad y costo esperado]
    F --> G[Guardar BORRADOR]
    G --> H[Llega proveedor]
    H --> I[Controlar recepcion]
    I --> J[Completar recibido real y vencimientos]
    J --> K[Confirmar recepcion]
    K --> L[Stock actualizado]
```
